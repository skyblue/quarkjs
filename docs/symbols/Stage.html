    Class Stage

    Extends DisplayObjectContainer.
    舞台是显示对象的根，所有显示对象都会被添加到舞台上，必须传入一个context使得舞台能被渲染。舞台是一种特殊显示对象容器，可以容纳子显示对象。

    paused
    指示舞台更新和渲染是否暂停。默认为false。
     
    stageX
